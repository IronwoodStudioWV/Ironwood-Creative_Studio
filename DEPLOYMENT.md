# Ironwood Creative Studio - Vercel Deployment Guide\n\n## Prerequisites\n\n- GitHub account with access to this repository\n- Vercel account (free tier available at https://vercel.com)\n- Node.js 18+ installed locally (for testing)\n\n## Step 1: Prepare Your Repository\n\n### 1a. Merge setup branch to website-development\n\nFirst, create a Pull Request to merge your setup branch:\n\n```bash\n# Local commands (after cloning)\ngit fetch origin\ngit checkout setup/initial-project-structure\n# Visit GitHub to create PR: setup/initial-project-structure → website-development\n```\n\n### 1b. Add Design Assets to Repository\n\nYour design assets need to be in the repository for Vercel to access them:\n\n```bash\n# Create public/images directory\nmkdir -p public/images\n\n# Copy your design files from website-development branch\ncp \"Ironwood logo  Idea.png\" public/images/ironwood-logo.png\ncp Ironwood_Facebook_Banner_1200x628.png public/images/facebook-banner.png\ncp \"Ironwood Business Card.png\" public/images/business-card.png\ncp 1770925224501.png public/images/design-mockup.png\n\n# Commit and push\ngit add public/images/\ngit commit -m \"Add design assets to public folder\"\ngit push origin setup/initial-project-structure\n```\n\n## Step 2: Deploy to Vercel\n\n### Option A: Vercel Dashboard (Recommended for Beginners)\n\n1. **Sign up/Login to Vercel**\n   - Go to https://vercel.com\n   - Click \"Sign Up\" and choose \"Continue with GitHub\"\n   - Authorize Vercel to access your repositories\n\n2. **Import Your Project**\n   - Click \"New Project\"\n   - Select `IronwoodStudioWV/Ironwood-Creative_Studio`\n   - Select branch: `setup/initial-project-structure`\n   - Framework: Next.js is auto-detected ✅\n   - Click \"Deploy\"\n\n3. **Configure Build Settings (Optional)**\n   - Build Command: `npm run build`\n   - Output Directory: `.next`\n   - Install Command: `npm install`\n   - Leave defaults and click \"Deploy\"\n\n### Option B: Vercel CLI (Advanced)\n\n```bash\n# Install Vercel CLI globally\nnpm install -g vercel\n\n# Login to Vercel\nvercel login\n\n# Deploy from your project directory\ncd Ironwood-Creative_Studio\nvercel\n\n# Follow the prompts:\n# - Set project name: ironwood-creative-studio\n# - Link to existing project: No\n# - Production scope: IronwoodStudioWV\n```\n\n## Step 3: Configure Custom Domain (Optional)\n\nYour current deployment URL is: `https://ironwood-creative-studio.vercel.app`\n\nTo use a custom domain:\n\n1. Go to your Vercel project dashboard\n2. Settings → Domains\n3. Add your domain (requires DNS configuration)\n4. Follow Vercel's DNS setup instructions\n\n## Step 4: Continuous Deployment\n\n**Your site will automatically redeploy when you:**\n- Push to `setup/initial-project-structure` branch\n- Merge PRs into that branch\n- Update environment variables\n\n## Environment Variables (If Needed)\n\nFor contact form backend integration:\n\n1. Go to your Vercel project → Settings → Environment Variables\n2. Add variables like:\n   ```\n   NEXT_PUBLIC_FORM_ACTION=https://api.yourdomain.com/contact\n   SENDGRID_API_KEY=your_key_here\n   ```\n\n## Troubleshooting\n\n### Build Fails\n- Check build logs in Vercel dashboard\n- Run locally: `npm run build` to test\n- Verify all dependencies in `package.json`\n\n### Images Not Showing\n- Confirm images are in `/public/images/`\n- Check file paths in components\n- Rebuild and redeploy\n\n### Site Still Shows \"Coming Soon\"\n- Clear browser cache: `Ctrl+Shift+Delete` (or Cmd+Shift+Delete on Mac)\n- Hard refresh: `Ctrl+F5`\n- Check if deployment completed successfully in Vercel dashboard\n\n## Monitoring & Analytics\n\nVercel provides free analytics:\n- Real-time request data\n- Performance metrics\n- Error tracking\n\nAccess in your Vercel project dashboard under \"Analytics\"\n\n## Next Steps\n\n1. **Deploy to Vercel** using steps above\n2. **Test your site** at the provided Vercel URL\n3. **Verify all components** render correctly:\n   - Header with logo\n   - Hero section with Facebook banner\n   - Services grid\n   - About section\n   - Contact form\n4. **Connect custom domain** if you have one\n5. **Set up form backend** (optional - currently shows alert only)\n\n## Rollback to Previous Version\n\nIf something goes wrong:\n\n1. Go to Vercel Dashboard → Deployments\n2. Find the previous working deployment\n3. Click the three dots → \"Promote to Production\"\n\n## Need Help?\n\n- Vercel Docs: https://vercel.com/docs\n- Next.js Docs: https://nextjs.org/docs\n- GitHub Issues: Create an issue in your repository\n\n---\n\n**Your site will be live at:**\n- `https://ironwood-creative-studio.vercel.app` (automatic)\n- Custom domain (after setup)\n"
